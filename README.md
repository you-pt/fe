# YOU-PT 당신을 위한 홈 트레이닝 서비스

## Introduction
- 웹 RTC를 활용한 실시간 PT 서비스
  - 실시간 채팅으로 자유로운 의사소통과 피드백
- AI 식단관리 서비스
- 운동 스케줄 관리 기능
